# Read the uomthesis: metadata block from a project's index.qmd

Read the uomthesis: metadata block from a project's index.qmd

## Usage

``` r
read_uomthesis_metadata(project_path)
```

## Arguments

- project_path:

  Path to project root (contains \_quarto.yml + index.qmd).

## Value

A list of the uomthesis block; errors if missing.
